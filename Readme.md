## Тема: Розробка статичного інтерфейсу Web-додатка

### Завдання роботи:
Розробити статичні Web-сторінки додатка та перевірити коректне їх відображення на екранах різного розміру. Основними інструментами розробки є мови HTML5, CSS3 та фреймворк Twitter Bootstrap.

**Обрана тематика:** Калькулятор (режим звичайний та для програміста).

### Функціональні можливості додатка:
- **Звичайний режим:** базові арифметичні операції для повсякденного використання.
- **Режим програміста:** підтримка систем числення (HEX, DEC, OCT, BIN) та побітових логічних операцій (AND, OR, XOR, NOT).
- **Адаптивність:** коректне відображення на мобільних пристроях, планшетах та ПК.
- **Система сторінок:** реєстрація, вхід, профіль користувача та сторінка "Про додаток".
